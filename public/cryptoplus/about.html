About CS50 CryptoPlus

This website is my final project for CS50x.

At the start of 2014 I enrolled onto a course called CS50x An Introduction to Computer Science which was offered through edX. I was looking for a really thorough introduction to the discipline which couldn't be had through traditional coding courses and this course really fit the bill. It will do me a world of good in my preparations to undertake an undergraduate degree in computer science this coming September.

The idea for this site came from Problem Set 2 in Week 2 of the course. We were asked to write two programs, one of which was to encrypt text into ciphertext using the Caesar Shift method and the other was to encrypt text into ciphertext using the Vig√®nere method. Relative to the stage in the course, they both presented a satisfying challenge which I wanted to explore further. Cryptography was a topic that for a long time I have found interesting, as it combines pure mathematics with modern-day computer science problems, such as the need for secure transmission of data over a network, so I decided that I would create something related to this field.

I primarily developed this site using HTML5, CSS and JavaScript as they are powerful modern web technologies which offer the flexibility to create a wide variety of applications.

I hope you enjoy using CS50 CryptoPlus!
